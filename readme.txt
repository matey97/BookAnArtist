Para hacer funcionar el proyecto hay que seguir los siguientes pasos:
  - Clonar el repositorio.
  - Abrir en distintos IDEs los proyectos en los directorios client y server.
    - En el cliente ejecutar la instrucción npm install para descargar todas las dependencias.
    - En el servidor las dependencias se descargarán automáticamente.
  - Ejecutar servidor y cliente:
    - Arrancar servidor como Spring Boot Application.
    - Arrancar el cliente con el comando npm start.
  - Acceder a la plataforma mediante http://localhost:4200.
